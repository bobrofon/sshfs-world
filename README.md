# sshfs-world
Build scripts for standalone sshfs with android support.
# Supported architectures
* arm
* i686

# How to use
Build arm and i686 binaries
```
$ mkdir -p build
$ cd build
$ cmake ..
$ make
$ cd -
```
Get result from ./stable
# Dependencies
* [CMake](https://cmake.org/)
